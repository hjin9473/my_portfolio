📌 Key Achievements (Troubleshooting)

  • AWS RDS 'Too many connections' 장애 해결:

    프리티어 RDS의 낮은 커넥션 제한(30개)으로 인한 서버 마비 현상 발생. HikariCP 설정을 최적화하고, RDS 파라미터 그룹을 수정하여 max_connections를 100으로 상향 조정해 안정성 확보

  • 외부 API 연동 시 병목(Blocking) 해결:

    국세청 API 호출 시 응답 지연으로 서버 스레드가 고갈되는 현상(Thread Starvation) 발생. ReadTimeout(3s) 설정을 명시적으로 적용하여 외부 장애가 내부 시스템으로 전이되지 않도록 회복 탄력성(Resilience) 확보
