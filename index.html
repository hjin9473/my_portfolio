GitHub >
Spring Boot 4 Spring Security Spring Data JPA AWS (EC2/RDS) Docker MariaDB

💡 기획 의도 & 아키텍처

대규모 트래픽을 고려한 확장 가능한 커머스 플랫폼입니다. Spring Boot 4.0 & Java 21 최신 환경을 도입했으며, Docker & AWS를 활용해 개발부터 배포까지의 환경 불일치를 해결했습니다.

📌 Key Achievements (Troubleshooting)

  • ✅ 데이터 무결성을 지킨 AWS 마이그레이션:

    로컬 Docker(MariaDB)에서 AWS RDS로 이관 중 Collation 충돌 및 트리거 권한 에러 발생. sed 스크립트로 덤프 파일을 정밀 치환하여 데이터 100% 이관 성공.

  • ✅ JPA & QueryDSL 도입으로 생산성 40% 향상:

    기존 MyBatis의 반복적인 SQL 작업 비효율을 개선하기 위해 JPA 도입. 복잡한 동적 쿼리(상품 검색)는 QueryDSL로 최적화하여 Type-Safety 확보.

  • ✅ Stateless 보안 아키텍처:

    Spring Security와 Redis Session을 연동하여, 서버 스케일 아웃 시에도 로그인이 유지되는 분산 환경 구축.
